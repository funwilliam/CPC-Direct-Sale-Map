## 1. 產品定位

台灣中油**直營**加油站地圖 PWA。核心價值：**幫使用者避開加盟站、快速找到直營站**。
對外品牌「順路加油」；非中油官方應用，資料來源為政府開放資料並於頁尾標示。

## 2. 目標使用者與情境

開車/騎車、只想在中油直營站加油的人。典型情境：行進間打開 app，3 秒內看到最近的直營站，一鍵導航過去；途中不誤入加盟站。

## 3. 功能需求

### F1 主頁地圖
- 讀取瀏覽器定位，顯示於內嵌互動地圖（Google Maps）；使用者位置以原生 Google Maps 等級的藍點呈現（白圈藍芯＋呼吸光暈）。
- 每次新 session 首次定位後自動調整視野：至少可見 1 間直營站；搜尋半徑上限 **15 km（≈20 分鐘車程）**，超出則以**固定美觀縮放（街區層級）**置中使用者並提示。僅自動執行一次，之後不搶使用者視角。
- **定位按鈕**（右下角圓鈕，原生地圖慣例）：點擊即回到我的位置並重新套用上述視野規則。
- 拒絕定位 → fallback 台灣全島視野。
- 效能門檻：地圖拖曳/縮放需流暢（marker 視窗裁剪，拖曳中不得增刪 DOM）。

### F2 加油站資訊卡
點擊直營站 marker 顯示：站名、地址（一鍵複製）、供應油品（92/95/98/超級柴油）、「Google Maps 導航」按鈕（deep link）。

### F3 地圖瀏覽
自由縮放/平移；視窗內直營站顯著標示（~626 站，用 clustering）。

### F4 加盟站防誤闖
加盟站（~1300 站）以次要視覺呈現（小尺寸、低透明度）；低縮放級別（zoom < 12）整層隱藏。點擊仍可看資訊卡，卡上明顯標示「加盟站」。

### F5 清單頁
- 模糊搜尋：站名、行政區（如「文山區」）。
- 篩選：92/95/98/超級柴油 油品。
- 排序：有定位時**依車程**（先以直線距離粗篩近站再查車程，控制 API 用量；車程不可用時退回直線距離）；搜尋中依相關性。
- 搜尋結果中加盟站一眼可辨（明顯的視覺標記）。

### F6 油價頁
- 當前中油牌價（92/95/98/超級柴油 + 生效日期）。
- 歷史油價走勢圖（Phase 2；資料自 Phase 0 起每週累積 + 官網歷史頁一次性回填）。

## 4. 非目標（明確不做）

- 不做其他品牌（台塑等）加油站。
- 不做會員/登入/個人化。
- 不做即時路況、油站排隊狀況。
- 不做 i18n（純繁體中文）。
- 「下週油價預告」：無官方資料源，列 backlog 待調查（浮動油價公式推估方案），**不進 MVP**。
- 離線地圖：Google Maps 圖磚依 ToS 不可快取；離線時清單頁/油價頁可用，地圖顯示不可用提示。

## 5. 成功標準

- 使用者從開啟 app 到看到最近直營站 ≤ 3 秒（4G 網路、非首次載入）。
- 視覺設計通過 `spec/design.md` 驗收門檻（商業級美學為硬性要求）。
- 直營/加盟判別 100% 與中油官網查詢系統一致（資料管線每次更新自動交叉驗證）。
- Lighthouse：PWA 可安裝；Performance ≥ 90（行動裝置模擬）。
- 月度 Google Maps 用量 < 免費額度 50%（防超量計費）。

## 7. 里程碑

Phase 0 地基（資料管線+文件+腳手架）→ Phase 1 MVP（F1-F5）→ Phase 2 油價（F6）→ Phase 3 打磨（效能/a11y/新鮮度標示）。
